Aplicativo para controle de entrada e saída do horário trabalho, podendo registrar:
Entrada, saída para intervalo, retorno e saída.



## Funcionalidades

- Cadastro de usuário
- Login / Logout
- Registro de Ponto
- Histórico de dias trabalhados
